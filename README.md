## 正确打开姿势

> 预备工作：请确保已安装 Node.js 在你的电脑。

1. 安装 [Start Server](https://github.com/xudafeng/startserver) 依赖。执行 `$ npm install`。
2. 启动它，执行 `$ npm start` 。
3. 在弹出的浏览器窗口中点击 `tasted.md` 即可。
4. 若非幻灯片形式，则勾选右上方的 `slide` 单选框。
